# Changelog

All notable changes to OneVoice (formerly RTranslator) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed
- Rebranded from RTranslator to OneVoice
- Updated all GitHub links to point to thekarampuri/Translate-Call repository
- Updated privacy policy links across all language variants
- Improved launcher icon with better contrast (light gray background)

### Fixed
- Fixed circular reference in launcher icon configuration
- Fixed blank logo display issue in adaptive icons
- Corrected branding inconsistencies in string resources

### Added
- Privacy policy translations for Ukrainian and Chinese
- Comprehensive project documentation

### Removed
- Temporary build and logo generation scripts
- Duplicate Gradle wrapper files
- Duplicate license file

## [2.1.4] - Previous Release

### Features
- Real-time conversation translation between multiple people
- WalkieTalkie mode for single-device translation
- Text translation mode
- Offline AI models (OpenAI Whisper + Meta NLLB)
- Bluetooth connectivity for multi-device conversations
- Support for 7+ languages

### Technical
- Minimum Android SDK: 24 (Android 7.0)
- Target Android SDK: 33 (Android 13)
- Requires 6GB+ RAM for optimal performance
